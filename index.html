Brandon Loveall: bringing ideas to life with a lot of fun along the way.

Full Stack - Backend - Frontend

Utah-based software developer with expertise in either end, with each one having its own unique and exciting challenges! No language or framework is safe from getting added to my skill set.

So just who am I?

My story began programming small personal projects on various platforms at a young age. I fell in love with everything that programming had to offer, from robotics to applications, and knew this passion would only continue to grow. Even when settled into a career, my desire to learn about computer science remains burning long after work is over.

Outside of programming, nature is my solitude. Growing up in San Diego gave me beautiful beaches and hiking trails to explore and you can always spot me going for walks in the rain or snow and appreciating the weather.

Conway's Game of Life

React | JS | HTML/CSS

Recreation of Conway's Game of life using the canvas HTML element! Created to practice creating graphics for uses such as graphs and advanced diagrams. Users can place cells on the grid and "step" through the simulation.

Chirper Social Media Site

React | JS | Node.js | Postgres

Challenging myself by recreating the popular social media site Twitter. Users can login with persistent session, sign up, tweet, like tweets, edit and delete their tweets, follow users, and customize their profile.

GitHub Demo

Country API Front End

React | JS | HTML/CSS

Recreated a front end for the Country API to meet exact user specifications based off diagrams. Users can filter countries based off name and region and click on a country for more information.
